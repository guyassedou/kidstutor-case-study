# Publication Checklist

## Repository boundary

- [ ] Repository contains Markdown and approved media only
- [ ] No production source code
- [ ] No copied private Git history or commit messages
- [ ] No database schemas or migrations
- [ ] No API or server-function implementation
- [ ] No internal prompts or complete agent instructions
- [ ] No production configuration or environment files
- [ ] No roadmap or monetization mechanics

## Privacy

- [ ] No child or parent names
- [ ] No account identifiers
- [ ] No real progress, usage, or behavioral data
- [ ] No raw feedback
- [ ] All examples are synthetic
- [ ] Every screenshot checked for hidden information
- [ ] Every video reviewed frame by frame

## Claims

- [ ] Technical evidence tied to a date
- [ ] No unsupported educational-outcome claims
- [ ] No unsupported production-usage claims
- [ ] AI involvement described accurately
- [ ] Proprietary status clearly stated

## Rights

- [ ] Proprietary notice included
- [ ] Third-party assets and trademarks reviewed
- [ ] Media ownership confirmed
- [ ] Required attributions present

## Quality

- [ ] Markdown links verified
- [ ] Mermaid diagrams render
- [ ] No broken media references
- [ ] Spelling and accessibility review complete
- [ ] Independent disclosure review complete

## Final approval

- [ ] Owner approves repository name and description
- [ ] Owner approves every public claim
- [ ] Owner approves every media asset
- [ ] Owner explicitly approves publication
